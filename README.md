VisionForge AI - Smart OCR System

Project Description

VisionForge AI is a Python-based Optical Character Recognition (OCR) system that extracts text from images and highlights detected text regions automatically.

The project uses OpenCV for image processing and Tesseract OCR for text recognition.

---

Features

- Extract text from images
- Highlight detected text
- Save extracted text to a file
- Save processed image with bounding boxes
- Beginner-friendly AI project

---

Technologies Used

- Python
- OpenCV
- Pytesseract
- NumPy

---

Installation

Install required libraries:

pip install -r requirements.txt

Install Tesseract OCR separately.

---

How to Run

Place your image inside:

input_images/sample.jpg

Run:

python main.py

---

Output

- extracted_text.txt
- processed_image.jpg
